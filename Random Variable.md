## Formal Definition

A **Random Variable** (RV) is a measurable function \(X: \Omega \to \mathbb{R}\) that maps outcomes from a [[sample space]] \(\Omega\) to real numbers, assigning a numerical value to each possible outcome of a random experiment. It provides a mathematical representation of uncertain quantities by translating abstract outcomes into quantitative form, enabling probabilistic analysis through distribution functions, moments, and other statistical descriptors.

## Mathematical Formulation

### General Form

Given a probability space \((\Omega, \mathcal{F}, P)\), a **random variable** is a function \(X: \Omega \to \mathbb{R}\) such that for every Borel set \(B \in \mathcal{B}(\mathbb{R})\):
$$
\{\omega \in \Omega: X(\omega) \in B\} \in \mathcal{F}
$$
This measurability condition ensures \(P(X \in B)\) is well-defined.

### Component Breakdown

**Pre-image Condition:** \(X^{-1}(B) \in \mathcal{F}\) for all Borel sets \(B\)
**Induced Probability Measure:** \(P_X(B) = P(X^{-1}(B))\) on \((\mathbb{R}, \mathcal{B}(\mathbb{R}))\)
**Distribution Function:** \(F_X(x) = P(X \leq x) = P(\{\omega: X(\omega) \leq x\})\)

### Special Cases
1. **Discrete Random Variable:**
   $$
   X(\Omega) = \{x_1, x_2, \dots\} \text{ (countable set)}
   $$
   Probability mass function: \(p_X(x_i) = P(X = x_i)\)

2. **Continuous Random Variable:**
   $$
   F_X(x) = \int_{-\infty}^x f_X(t) dt
   $$
   where \(f_X\) is [[Probability Density Function (PDF)]]

3. **Mixed Random Variable:**
   Combination of discrete and continuous components

## Fundamental Principles

### Core Properties
1. **Measurability:** Ensures probabilities of events involving \(X\) are defined
2. **Induced Distribution:** \(P_X\) is a valid probability measure on \(\mathbb{R}\)
3. **Function Composition:** If \(g: \mathbb{R} \to \mathbb{R}\) is Borel measurable, then \(g(X)\) is also a random variable

### Distribution Characterization
- **[[Cumulative Distribution Function (CDF)]] (CDF):** \(F_X(x) = P(X \leq x)\)
- **Properties:** Non-decreasing, right-continuous, \(\lim_{x \to -\infty} F_X(x) = 0\), \(\lim_{x \to \infty} F_X(x) = 1\)
- **Survival Function:** \(S_X(x) = 1 - F_X(x) = P(X > x)\)

## Theoretical Framework

### Measure-Theoretic Foundation
Random variables are \(\mathcal{F}/\mathcal{B}(\mathbb{R})\)-measurable functions, linking abstract outcomes to numerical observations.

### σ-Algebra Generated by a Random Variable
The smallest σ-algebra with respect to which \(X\) is measurable:
$$
\sigma(X) = \{X^{-1}(B): B \in \mathcal{B}(\mathbb{R})\} \subseteq \mathcal{F}
$$
This represents the "information" contained in \(X\).

### Transformations and Functions
If \(g: \mathbb{R} \to \mathbb{R}\) is Borel measurable, then \(Y = g(X)\) is a random variable with:
- Distribution: \(F_Y(y) = P(g(X) \leq y)\)
- Expectation: \(E[g(X)] = \int_\Omega g(X(\omega)) dP(\omega) = \int_\mathbb{R} g(x) dF_X(x)\)

## Classification and Variants

### By Type of Support
1. **Discrete:** Finite or countably infinite range (Bernoulli, Binomial, Poisson, Geometric)
2. **Continuous:** Uncountable range with density (Normal, Exponential, Uniform)
3. **Singular Continuous:** Continuous but not absolutely continuous (Cantor distribution)
4. **Mixed:** Combination of discrete and continuous parts

### By Dimensionality
1. **Univariate:** Single random variable \(X\)
2. **Multivariate:** Random [[Vector]] \(\mathbf{X} = (X_1, \dots, X_n)\)
3. **Random Elements:** Values in more general spaces (functions, sets, etc.)

### By Dependence Structure
1. **Independent:** \(P(X \in A, Y \in B) = P(X \in A)P(Y \in B)\)
2. **Correlated:** Non-zero [[covariance]]/dependence
3. **Exchangeable:** Joint distribution invariant under permutations

### Special Classes
1. **Indicator Variables:** \(I_A(\omega) = 1\) if \(\omega \in A\), 0 otherwise
2. **Simple Random Variables:** Finite range
3. **Random Variables with Moments:** \(E[|X|^p] < \infty\) for some \(p > 0\)

## Derivation and Proof

### Construction from Probability Space

**Step 1: Define measurable function**
Given \((\Omega, \mathcal{F}, P)\), choose \(X: \Omega \to \mathbb{R}\) satisfying measurability.

**Step 2: Induced measure**
For \(B \in \mathcal{B}(\mathbb{R})\), define:
$$
P_X(B) = P(\{\omega: X(\omega) \in B\})
$$

**Step 3: Verify properties**
- \(P_X(\emptyset) = P(X^{-1}(\emptyset)) = P(\emptyset) = 0\)
- \(P_X(\mathbb{R}) = P(X^{-1}(\mathbb{R})) = P(\Omega) = 1\)
- Countable additivity follows from properties of \(P\)

### Existence and Construction
- **For any CDF F:** There exists a probability space and random variable with CDF = F (Skorokhod representation)
- **For given distribution:** Can construct on \((\mathbb{R}, \mathcal{B}(\mathbb{R}), \mu)\) with \(X(x) = x\) (identity)

### Transformation Proof
**Theorem:** If \(X\) is RV and \(g\) Borel measurable, then \(g(X)\) is RV.

**Proof:** For \(B \in \mathcal{B}(\mathbb{R})\), 
$$
(g \circ X)^{-1}(B) = X^{-1}(g^{-1}(B))
$$
Since \(g^{-1}(B) \in \mathcal{B}(\mathbb{R})\) and \(X\) measurable, \(X^{-1}(g^{-1}(B)) \in \mathcal{F}\).

**Discrete Random Variable:**

import numpy as np
from collections import defaultdict

class DiscreteRandomVariable:
    def __init__(self, values, probabilities):
        self.values = np.array(values)
        self.probabilities = np.array(probabilities)
        assert np.allclose(self.probabilities.sum(), 1.0)
        
    def expectation(self, g=None):
        if g is None:
            return np.sum(self.values * self.probabilities)
        else:
            return np.sum(g(self.values) * self.probabilities)
    
    def variance(self):
        mu = self.expectation()
        return np.sum((self.values - mu)**2 * self.probabilities)
    
    def sample(self, n=1):
        return np.random.choice(self.values, size=n, p=self.probabilities)

# Bernoulli(p)
bernoulli = DiscreteRandomVariable([0, 1], [0.7, 0.3])

# Binomial(n, p) approximation
n, p = 10, 0.5
values = list(range(n+1))
probs = [np.math.comb(n, k) * p**k * (1-p)**(n-k) for k in values]
binomial = DiscreteRandomVariable(values, probs)

**Continuous Random Variable:**
```python
class ContinuousRandomVariable:
    def __init__(self, pdf=None, cdf=None, ppf=None):
        self.pdf = pdf  # [[Probability density function]]
        self.cdf = cdf  # [[Cumulative distribution function]]
        self.ppf = ppf  # Percent point function (inverse CDF)
    
    def expectation(self, g=None, n_samples=10000):
        # Monte Carlo integration
        samples = self.sample(n_samples)
        if g is None:
            return np.mean(samples)
        else:
            return np.mean(g(samples))
    
    def sample(self, n=1):
        if self.ppf:
            # Inverse transform sampling
            u = np.random.uniform(0, 1, n)
            return self.ppf(u)
        else:
            raise NotImplementedError("Sampling requires inverse CDF")
    
    def probability(self, a, b):
        if self.cdf:
            return self.cdf(b) - self.cdf(a)
        else:
            # Numerical integration of PDF
            from scipy import integrate
            return integrate.quad(self.pdf, a, b)[0]

# Normal distribution approximation
from scipy.stats import norm
normal = ContinuousRandomVariable(
    pdf=norm.pdf,
    cdf=norm.cdf,
    ppf=norm.ppf
)
```

### Numerical Methods
1. **Inverse transform sampling:** For continuous RVs with known inverse CDF
2. **Acceptance-rejection:** For RVs with complex densities
3. **Markov Chain Monte Carlo:** For high-dimensional or complex distributions
4. **Transform methods:** Using relationships between distributions

### Memory and Performance
- **Discrete:** Store values and probabilities (memory ~ O(n))
- **Continuous:** Store function pointers or parameters (memory ~ O(1))
- **Sampling:** Computational cost varies by method

## Advantages and Limitations

### Strengths
1. **Mathematical tractability:** Enables calculus and analysis of uncertainty
2. **Unified framework:** Handles discrete, continuous, and mixed variables
3. **Rich theory:** Well-developed theory of moments, transforms, limits
4. **Computational implementation:** Easy to simulate and analyze

### Weaknesses
1. **Measurability requirements:** Technical condition can be restrictive
2. **Infinite moments:** Some RVs lack mean or variance
3. **Heavy-tailed distributions:** Can challenge statistical methods
4. **Dependence modeling:** Multivariate dependence can be complex

### Validity Range
- **Appropriate:** When outcomes can be meaningfully quantified
- **Limited usefulness:** For purely qualitative outcomes without natural ordering
- **Challenging:** For RVs with pathological distributions (e.g., Cauchy distribution for mean estimation)

## Advanced Topics

### Convergence Concepts
1. **Almost sure convergence:** \(P(\lim_{n\to\infty} X_n = X) = 1\)
2. **Convergence in probability:** \(\lim_{n\to\infty} P(|X_n - X| > \epsilon) = 0\)
3. **Convergence in distribution:** \(\lim_{n\to\infty} F_{X_n}(x) = F_X(x)\) at continuity points
4. **\(L^p\) convergence:** \(\lim_{n\to\infty} E[|X_n - X|^p] = 0\)

### Characteristic Functions
[[Fourier Transform]] of distribution:
$$
\phi_X(t) = E[e^{itX}] = \int_{-\infty}^{\infty} e^{itx} dF_X(x)
$$
Uniquely determines distribution, useful for limit theorems.

### Stochastic Processes
Random variables indexed by time/space:
- **Discrete-time:** \(X_1, X_2, \dots\)
- **Continuous-time:** \(\{X_t: t \geq 0\}\)

### Random Vectors and Matrices
Multidimensional extensions:
- **[[Covariance]] structure:** Σ = Cov(X)
- **Joint distributions:** Multivariate normal, copulas
- **Random matrices:** Eigenvalue distributions

## Special Cases and Examples

### Common Distributions
1. **Normal/Gaussian:** \(X \sim N(\mu, \sigma^2)\)
   - PDF: \(f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)\)
   - Properties: Symmetric, light-tailed, stable under addition

2. **Exponential:** \(X \sim \text{Exp}(\lambda)\)
   - PDF: \(f(x) = \lambda e^{-\lambda x}\) for \(x \geq 0\)
   - Memoryless property: \(P(X > s+t | X > s) = P(X > t)\)

3. **Poisson:** \(X \sim \text{Poisson}(\lambda)\)
   - PMF: \(P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}\) for \(k=0,1,2,\dots\)
   - Models counts of rare events

### Interesting Edge Cases
1. **Cauchy Distribution:** No finite mean or variance
2. **Bernoulli with p=0 or 1:** Degenerate (deterministic) random variable
3. **Mixture distributions:** \(X \sim pF_1 + (1-p)F_2\)
4. **Singular continuous:** Cantor distribution (CDF continuous but not absolutely continuous)

## Error Analysis

### Approximation Errors
1. **Discretization error:** Approximating continuous RV with discrete grid
2. **Truncation error:** Ignoring tail probabilities
3. **Monte Carlo error:** \(O(1/\sqrt{n})\) for sample averages

### Numerical Stability
- **Variance calculation:** Use \(E[X^2] - (E[X])^2\) with care for cancellation
- **Log probabilities:** Work in log-space to avoid underflow
- **Importance sampling:** Reduce variance of Monte Carlo estimates

### Model Misspecification
- **Wrong distributional assumption:** Using normal for heavy-tailed data
- **Ignoring dependence:** Treating correlated variables as independent
- **Parameter uncertainty:** Estimation error in distribution parameters

## Relationship to Other Concepts

### Connection to Expectation
Expectation \(E[X] = \int_\Omega X dP\) is the Lebesgue integral of \(X\) with respect to \(P\).

### Relation to Distribution Function
CDF \(F_X\) completely characterizes distribution of \(X\):
- **Kolmogorov's theorem:** For any right-continuous, non-decreasing \(F\) with limits 0 and 1, there exists RV with CDF = \(F\)

### Contrast with Deterministic Variable
- **Random variable:** Value determined by chance (with distribution)
- **Deterministic variable:** Fixed, known value

### Link to Statistics
- **Sample:** Realizations \(x_1, \dots, x_n\) of i.i.d. random variables
- **Statistic:** Function of sample random variables (e.g., sample mean \(\bar{X}_n\))
- **Estimator:** Statistic used to estimate population parameter

# Abstract base class for random variables
class RandomVariable(ABC):
    """Abstract base class for random variables"""
    
    @abstractmethod
    def sample(self, n=1):
        """Generate n samples from the distribution"""
        pass
    
    @abstractmethod
    def cdf(self, x):
        """[[Cumulative distribution function]]"""
        pass
    
    @abstractmethod
    def pdf(self, x):
        """Probability density/mass function"""
        pass
    
    def expectation(self, g=None, method='analytic', n_samples=10000):
        """Compute expectation E[g(X)]"""
        if method == 'analytic' and hasattr(self, '_expectation'):
            if g is None:
                return self._expectation
            else:
                # May need numerical integration
                pass
        
        # Monte Carlo fallback
        samples = self.sample(n_samples)
        if g is None:
            return np.mean(samples)
        else:
            return np.mean(g(samples))
    
    def variance(self, method='analytic', n_samples=10000):
        """Compute variance Var(X)"""
        if method == 'analytic' and hasattr(self, '_variance'):
            return self._variance
        
        # Monte Carlo
        samples = self.sample(n_samples)
        return np.var(samples)
    
    def plot_pdf(self, x_range=None, n_points=1000):
        """Plot PDF/PMF"""
        if x_range is None:
            # Default range based on distribution
            if hasattr(self, 'mean') and hasattr(self, 'std'):
                mean = self.mean()
                std = self.std()
                x_range = (mean - 4*std, mean + 4*std)
            else:
                # Sample to estimate range
                samples = self.sample(1000)
                x_range = (np.min(samples), np.max(samples))
        
        x = np.linspace(x_range[0], x_range[1], n_points)
        y = self.pdf(x)

# Discrete random variable implementation
class DiscreteRV(RandomVariable):
    """Discrete random variable with finite support"""
    
    def __init__(self, values, probabilities):
        self.values = np.array(values)
        self.probabilities = np.array(probabilities)
        
        # Normalize probabilities
        self.probabilities = self.probabilities / self.probabilities.sum()
        
        # Precompute expectation and variance if possible
        self._expectation = np.sum(self.values * self.probabilities)
        self._variance = np.sum((self.values - self._expectation)**2 * self.probabilities)
    
    def sample(self, n=1):
        return np.random.choice(self.values, size=n, p=self.probabilities)
    
    def pdf(self, x):
        """Probability mass function"""
        result = np.zeros_like(x, dtype=float)
        for val, prob in zip(self.values, self.probabilities):
            result[x == val] = prob
        return result
    
    def cdf(self, x):
        """[[Cumulative distribution function]]"""
        result = np.zeros_like(x, dtype=float)
        for val, prob in zip(self.values, self.probabilities):
            result[x >= val] += prob
        return np.clip(result, 0, 1)
    
    def mean(self):
        return self._expectation
    
    def std(self):
        return np.sqrt(self._variance)

# Continuous random variable implementation
class ContinuousRV(RandomVariable):
    """Continuous random variable defined by PDF"""
    
    def __init__(self, pdf_func, support=(-np.inf, np.inf), 
                 cdf_func=None, sampling_func=None):
        self.pdf_func = pdf_func
        self.support = support
        self.cdf_func = cdf_func
        self.sampling_func = sampling_func
        
        # Precompute norm constant if needed
        self.norm_constant = 1.0  # Assume normalized
        
        # Try to compute mean and variance numerically
        try:
            self._expectation = integrate.quad(
                lambda x: x * self.pdf_func(x),
                self.support[0], self.support[1]
            )[0]
            
            self._variance = integrate.quad(
                lambda x: (x - self._expectation)**2 * self.pdf_func(x),
                self.support[0], self.support[1]
            )[0]
        except:
            self._expectation = None
            self._variance = None
    
    def sample(self, n=1):
        if self.sampling_func:
            return self.sampling_func(n)
        
        # Inverse transform sampling if CDF available
        elif self.cdf_func and hasattr(self, 'ppf_func'):
            u = np.random.uniform(0, 1, n)
            return self.ppf_func(u)
        
        # Rejection sampling as fallback
        else:
            return self._rejection_sampling(n)
    
    def _rejection_sampling(self, n):
        """Basic rejection sampling"""
        a, b = self.support
        if not np.isfinite(a) or not np.isfinite(b):
            # Estimate range from PDF mode
            # Simple implementation - in practice need better
            a, b = -10, 10
        
        # Find maximum of PDF
        x_vals = np.linspace(a, b, 1000)
        y_vals = self.pdf_func(x_vals)
        M = np.max(y_vals) * 1.1  # Add 10% margin
        
        samples = []
        while len(samples) < n:
            # Propose candidate
            x_prop = np.random.uniform(a, b)
            y_prop = np.random.uniform(0, M)
            
            # Accept if y_prop < pdf(x_prop)
            if y_prop < self.pdf_func(x_prop):
                samples.append(x_prop)
        
        return np.array(samples)
    
    def pdf(self, x):
        return self.pdf_func(x)
    
    def cdf(self, x):
        if self.cdf_func:
            return self.cdf_func(x)
        else:
            # Numerical integration
            result = np.zeros_like(x, dtype=float)
            for i, xi in enumerate(x):
                result[i] = integrate.quad(
                    self.pdf_func, self.support[0], xi
                )[0]
            return np.clip(result, 0, 1)
    
    def mean(self):
        if self._expectation is not None:
            return self._expectation
        else:
            return self.expectation()
    
    def std(self):
        if self._variance is not None:
            return np.sqrt(self._variance)
        else:
            return np.sqrt(self.variance())

# Specific distribution implementations
class NormalRV(ContinuousRV):
    """[[Normal distribution]] N(mu, sigma^2)"""
    
    def __init__(self, mu=0, sigma=1):
        self.mu = mu
        self.sigma = sigma
        
        # Define functions
        pdf_func = lambda x: (1/(sigma*np.sqrt(2*np.pi))) * \
                            np.exp(-0.5*((x-mu)/sigma)**2)
        
        cdf_func = lambda x: 0.5 * (1 + stats.erf((x-mu)/(sigma*np.sqrt(2))))
        
        # Inverse CDF (ppf) using scipy for sampling
        ppf_func = lambda u: mu + sigma * np.sqrt(2) * stats.erfinv(2*u - 1)
        
        sampling_func = lambda n: np.random.normal(mu, sigma, n)
        
        super().__init__(
            pdf_func=pdf_func,
            support=(-np.inf, np.inf),
            cdf_func=cdf_func,
            sampling_func=sampling_func
        )
        
        # Set exact moments
        self._expectation = mu
        self._variance = sigma**2

class ExponentialRV(ContinuousRV):
    """Exponential distribution Exp(lambda)"""
    
    def __init__(self, lam=1):
        self.lam = lam
        
        pdf_func = lambda x: lam * np.exp(-lam*x) if x >= 0 else 0
        
        cdf_func = lambda x: 1 - np.exp(-lam*x) if x >= 0 else 0
        
        # Inverse CDF: F^{-1}(u) = -ln(1-u)/λ
        ppf_func = lambda u: -np.log(1-u) / lam
        
        # Sampling via inverse transform
        sampling_func = lambda n: -np.log(1-np.random.uniform(0, 1, n)) / lam
        
        super().__init__(
            pdf_func=lambda x: pdf_func(x),
            support=(0, np.inf),
            cdf_func=lambda x: cdf_func(x),
            sampling_func=sampling_func
        )
        
        self._expectation = 1/lam
        self._variance = 1/(lam**2)

# Function transformations
def transform_random_variable(X, g):
    """Create new random variable Y = g(X)"""
    
    if isinstance(X, DiscreteRV):
        # For discrete, compute new values and probabilities
        new_values = []
        new_probs = []
        
        # Map old values through g
        value_prob_map = {}
        for val, prob in zip(X.values, X.probabilities):
            new_val = g(val)
            if new_val in value_prob_map:
                value_prob_map[new_val] += prob
            else:
                value_prob_map[new_val] = prob
        
        new_values = list(value_prob_map.keys())
        new_probs = list(value_prob_map.values())
        
        return DiscreteRV(new_values, new_probs)
    
    elif isinstance(X, ContinuousRV):
        # For continuous, need to compute distribution of g(X)
        # This is non-trivial in general
        # Simplified approach: define through sampling
        class TransformedRV(ContinuousRV):
            def __init__(self, X, g):
                self.X = X
                self.g = g
                
                # Sampling method
                sampling_func = lambda n: g(X.sample(n))
                
                # Approximate PDF using kernel density estimation
                # from samples (this is approximate)
                samples = sampling_func(10000)
                from scipy.stats import gaussian_kde
                kde = gaussian_kde(samples)
                
                super().__init__(
                    pdf_func=kde.pdf,
                    support=(np.min(samples), np.max(samples)),
                    sampling_func=sampling_func
                )
        
        return TransformedRV(X, g)
    
    else:
        raise TypeError(f"Unsupported random variable type: {type(X)}")

            self.norm2 = NormalRV(1, 1)

                n2 = n - n1
                samples1 = self.norm1.sample(n1)
