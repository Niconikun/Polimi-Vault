The **position** of a point particle is a fundamental vector quantity that specifies its location relative to a fixed reference point, called the origin, within a chosen coordinate system and [[Frame of Reference]].

To express this [[Vector]] in a component form amenable to Einstein notation, we first define a set of basis vectors. In an n-dimensional space (commonly $n=3$ for physical space), we choose a set of nn linearly independent basis vectors, ${e_1,e_2,...,e_n}$. The position vector r$ can then be written as a linear combination of these basis vectors:
$$
r=r^1e_1+r^2e_2+⋯+r^ne_n
$$
where $r^1,r^2,...,r^n$ are the **components** of the vector in this basis. These components are the coordinates of the point (e.g., x,y,zx,y,z in a Cartesian system).

### Definition using Einstein Notation

The Einstein summation convention states that whenever an index variable appears twice in a single term, once in an upper (superscript) and once in a lower (subscript) position, it implies summation over all possible values of that index.

Using this convention, the definition of the position vector is expressed with supreme conciseness:

$$
{\mathbf{r} = r^i \mathbf{e}_i}
$$

**Explanation of the Notation:**

*  $r$: The position vector.
*  $i$: A dummy index which runs from 1 to \(n\) (the dimension of the space).
*   $r^i$: The $i$-th component of the vector. The superscript $i$ is an index, not an exponent.
*   $\mathbf{e}_i$: The $i$-th basis vector.
*   **The Rule Applied:** The index $i$ appears twice in the term $r^i \mathbf{e}_i$, once as a superscript and once as a subscript. Therefore, according to the Einstein convention, this is shorthand for the sum $\sum_{i=1}^n r^i \mathbf{e}_i$.
